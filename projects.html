Project One

This project is a visual knowledge graph I built in Obsidian to map pharmacological relationships across 47 different substances. Each node represents a compound, and the connections illustrate links such as receptor interactions, structural similarities, and metabolic pathways. By visualizing this network, I created a tool for quickly identifying shared pharmacological profiles and exploring patterns that would otherwise remain buried in text-based research.
